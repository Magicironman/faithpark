# Spiritual Module (箴言 / 詩篇 / 與主同行)

## Modes
- 今日箴言（智慧）
- 今日詩篇（平安/安慰）
- 與主同行（經文 + 鼓勵 + 今日提醒）

## Mood Categories
- 平安, 焦慮, 鼓勵, 智慧, 信心, 疲倦, 壓力, 盼望, 感恩

## Flow
Tap card OR agent asks → pick category → show verse → Cantonese TTS → short encouragement

## Example
- 經文：耶和華是我的牧者…
- 鼓勵：今日主仍看顧你
- 提醒：停一停，交託
