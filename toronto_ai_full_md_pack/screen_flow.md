# Screen Flow

- Park Now
- Find My Car
- Active Timer
- Today’s Tasks
- Play News
- 今日箴言 / 今日詩篇 / 與主同行
- 🎤 Voice

## Parking
- Save GPS, notes, photo
- Set duration or end time
- Show countdown

## News
- Play 10-min Cantonese
- Category chips (Toronto/Canada/Finance/Tech)
- Source list

## Spiritual
- Cards: 今日箴言 / 今日詩篇 / 與主同行
- Mood chips (平安/焦慮/信心/智慧/鼓勵)
- Play Cantonese

## Voice
- Tap mic → command → route to module

## Settings
- Language (EN / 粵語)
- TTS voice
- Notification prefs
- News duration
- Daily quote time
