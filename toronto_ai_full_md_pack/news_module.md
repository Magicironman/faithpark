# News Module (Cantonese ~10 min)

## Sources (RSS)
- CityNews Toronto
- CBC
- Global News
- BNN Bloomberg
- TechCrunch / The Verge

## Flow
1. Fetch 3–5 headlines per category
2. Deduplicate + clean
3. Summarize bullets
4. Convert to Cantonese spoken script
5. TTS playback

## Order
- Toronto (1.5m)
- Canada (3m)
- Finance (2m)
- Tech (2m)
- Recap (1.5m)

## UI
- Play / Pause / Skip section
- Show sources (title, source, time)
