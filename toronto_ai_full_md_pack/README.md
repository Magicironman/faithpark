# Toronto AI Parking Assistant

Smart mobile app (Flutter) combining:
- Parking save + timer reminders
- Find my car (map)
- Daily tasks
- Cantonese + English voice agent
- 10-min Cantonese news briefing (Toronto/Canada)
- Spiritual encouragement (箴言 / 詩篇 / 與主同行)

## Run
```
flutter pub get
flutter run
```

## Structure
- /lib (Flutter code)
- /docs (product + modules)
- AGENTS.md (instructions for AI/Codex)
